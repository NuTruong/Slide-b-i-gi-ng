# 🎓 Slide Bài Giảng - Tự Động Hóa Tạo Slide Giáo Dục

## 📌 Giới Thiệu
Dự án này nhằm **tự động hóa quá trình tạo slide bài giảng** cho giáo viên từ các dữ liệu có sẵn, giúp tiết kiệm thời gian và tăng hiệu quả giảng dạy.

## 🎯 Mục Tiêu
- ✅ Chuyển đổi nội dung từ các định dạng dữ liệu (JSON, Markdown, CSV) thành slide PowerPoint
- ✅ Tạo template slide tự động với định dạng chuẩn hóa
- ✅ Hỗ trợ thêm hình ảnh, biểu đồ và nội dung đa phương tiện
- ✅ Giảm thiểu công sức thủ công cho giáo viên

## 🚀 Tính Năng (Dự Kiến)
- [ ] Đọc dữ liệu bài giảng từ file (JSON/Markdown/CSV)
- [ ] Phân tích nội dung và cấu trúc bài giảng
- [ ] Tạo slide PowerPoint tự động
- [ ] Áp dụng template và theme nhất quán
- [ ] Export slide ở nhiều định dạng (PPTX, PDF, HTML)
- [ ] Hỗ trợ thêm media (hình ảnh, video)

## 📁 Cấu Trúc Dự Án
```
Slide-b-i-gi-ng/
├── README.md
├── data/                  # Dữ liệu bài giảng
├── templates/             # Template slide
├── src/                   # Code chính
├── output/                # Slide được tạo
└── config/                # Cấu hình
```

## 🛠️ Công Nghệ Sử Dụng
- **Python** (chính)
- `python-pptx` - Tạo PowerPoint
- `pandas` - Xử lý dữ liệu
- `Pillow` - Xử lý hình ảnh
